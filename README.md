## Knowledge Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/2b111df893cd4a5ea27e4478111ed086/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Article Information
5. Material Download
6. Q&A
7. Forum Communication
8. Message Feedback
9. Personal Center
10. My Posts
11. My Favorites


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. User Management
4. Article Category Management
5. Article Information Management
6. Material Category Management
7. Material Download Management
8. Q&A Management
9. Forum Communication
10. Message Board Management
11. System Management
